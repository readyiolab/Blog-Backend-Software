## 📊 File Organization

### app.js (Express Server)
- Server setup
- Middleware configuration
- Route mounting
- Error handling

### models/database.js
- Database connection
- CRUD operations
- RBAC methods
- Query helpers

### middleware/rbac_middleware.js
- JWT verification
- Permission checking
- Role-based access
- Ownership verification

### controllers/ (Business Logic)
Separated by domain:
- **articleController.js** - All article operations
- **authController.js** - Authentication operations
- **categoryController.js** - Category operations
- **commentController.js** - Comment operations
- **userController.js** - User management

### routes/ (API Endpoints)
Separated by domain:
- **article_routes.js** - Article endpoints
- **auth_routes.js** - Auth endpoints
- **category_routes.js** - Category endpoints
- **comment_routes.js** - Comment endpoints
- **user_routes.js** - User endpoints

## 📝 How to Add New Features

### Example: Add Comment Management

1. **Create controller** - `controllers/commentController.js`
   ```javascript
   exports.getComments = async (req, res) => { ... };
   exports.createComment = async (req, res) => { ... };
   exports.deleteComment = async (req, res) => { ... };
   exports.approveComment = async (req, res) => { ... };
   ```

2. **Create routes** - `routes/comment_routes.js`
   ```javascript
   const commentController = require('../controllers/commentController');
   
   router.get('/article/:articleId', commentController.getComments);
   router.post('/', verifyToken, commentController.createComment);
   router.delete('/:id', verifyToken, commentController.deleteComment);
   router.put('/:id/approve', verifyToken, authorize('manage_comments'), commentController.approveComment);
   ```

3. **Mount in app.js**
   ```javascript
   const commentRoutes = require('./routes/comment_routes');
   app.use('/api/comments', commentRoutes);
   ```
