🚀 QUICK START (5 MINUTES)

1. Install dependencies
   npm install

2. Configure database
   cp .env.example .env
   nano .env

3. Import schema
   mysql -u root -p news_website_db < database_schema.sql

4. Run server
   npm start

5. Test
   curl http://localhost:5000/health
